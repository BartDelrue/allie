Navigation drawer

requirements

  • Keyboard trap for tab and shift-tab
  • Enable escape to close
  • Change focus upon opening
  • Place focus back to the trigger after closing
  • Set other content as aria-hidden=true while open
  • Can be triggered by a button or by swiping from the left
  • The button must communicate the state of the menu
  • The button must have a label

example

Preferably always use accompany an icon with text
If this is not possible, don't forget the aria-label attribute!

options available:

  • allie.naviDrawer.setDuration(seconds) default = 0.5
  • allie.naviDrawer.isSmooth(true)
  • allie.naviDrawer.enableSwipe(true, threshold) default = false, 480

Elements specifiek to the modal component receive a clear red border on focus.
This is for demonstration purposes.
